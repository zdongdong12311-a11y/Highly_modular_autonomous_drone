# 航点文件
# 格式: x y z hover_time
#   x         - 目标点 X 坐标 (米, map 坐标系)
#   y         - 目标点 Y 坐标 (米, map 坐标系)
#   z         - 目标高度 (米, 相对起飞点)
#   hover_time - 到达后悬停时间 (秒)
#
# 注意:
#   - 以 # 开头的行为注释，会被跳过
#   - 空行会被跳过
#   - 坐标单位为米

# 示例: 矩形航线 (顺时针)
0.0  0.0  0.6  2.0
1.0  0.0  0.8  3.0
1.0  1.0  1.0  3.0
0.0  1.0  0.8  3.0
0.0  0.0  0.6  2.0
